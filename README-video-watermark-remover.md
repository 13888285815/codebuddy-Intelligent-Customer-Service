# 视频去水印工具

这是一个基于浏览器的视频去水印工具，无需安装任何软件，可以直接在浏览器中使用。

## 功能特点

- 🎬 **支持多种格式**: MP4, WebM, AVI 格式视频
- 🎯 **精确区域选择**: 交互式选择水印区域
- 🎨 **多级处理算法**: 5种不同强度的去水印算法
- 📊 **两种处理模式**: 普通模式和逐帧模式
- 🖥️ **实时预览**: 处理前后实时对比
- 📥 **直接下载**: 处理结果可直接下载

## 使用方法

### 1. 上传视频
- 点击"选择视频文件"按钮或拖拽视频到上传区域
- 支持最大100MB的视频文件

### 2. 选择水印区域
- 点击"选择水印区域"按钮
- 在视频预览区域拖动鼠标选择水印位置
- 再次点击"完成选择"结束选择

### 3. 配置处理选项
- **处理强度**: 1-5级，从轻微到深度处理
- **处理模式**: 
  - 普通模式：快速处理整个视频
  - 逐帧模式：逐帧精确处理

### 4. 开始处理
- 点击"开始处理"按钮
- 等待处理完成（根据视频大小和处理模式不同，时间会有所差异）

### 5. 下载结果
- 处理完成后，"下载结果"按钮会变为可用
- 点击下载处理后的视频截图

## 处理算法说明

### 5级处理强度：

1. **轻度模糊** (强度1)
   - 轻微模糊水印区域
   - 适用于简单、半透明的水印

2. **中等模糊** (强度2)
   - 中等程度模糊
   - 适用于中等透明度的水印

3. **内容感知填充** (强度3)
   - 智能分析周围像素并填充
   - 适用于复杂背景的水印

4. **智能修复** (强度4)
   - 基于纹理匹配的修复
   - 适用于纹理背景的水印

5. **深度修复** (强度5)
   - 结合边缘检测和纹理合成
   - 适用于复杂、大面积的顽固水印

## 技术说明

### 前端技术
- HTML5 Canvas 用于视频渲染和处理
- JavaScript 实现图像处理算法
- CSS3 实现现代UI界面
- 本地文件API处理视频上传

### 算法原理
1. **像素采样**: 从水印区域周围的像素采样
2. **颜色匹配**: 分析颜色分布和纹理模式
3. **内容合成**: 基于采样结果合成新的像素
4. **边缘保持**: 识别并保持图像边缘
5. **纹理融合**: 将新像素与原始背景融合

### 性能优化
- 使用Canvas进行GPU加速渲染
- 优化的像素处理算法
- 渐进式处理，避免界面卡顿
- 智能内存管理

## 限制说明

1. **水印类型**: 
   - 最适用于固定位置、不透明的静态水印
   - 对动态水印效果有限
   - 对半透明水印效果较好

2. **处理时间**:
   - 视频时长越长，处理时间越长
   - 逐帧模式比普通模式耗时更长
   - 大文件可能需要较长时间

3. **输出格式**:
   - 当前版本输出为PNG图片格式
   - 后续版本可考虑支持视频输出

## 隐私与安全

- 所有处理都在浏览器本地完成
- 视频文件不会上传到任何服务器
- 处理完成后自动清理临时文件
- 保证用户隐私和数据安全

## 常见问题

### Q: 为什么处理后的效果不理想？
A: 可以尝试调整以下设置：
   - 增加处理强度
   - 切换到逐帧模式
   - 重新选择更精确的水印区域

### Q: 支持多大的视频文件？
A: 最大支持100MB的视频文件，更大的文件可能导致浏览器内存不足。

### Q: 为什么下载的是图片而不是视频？
A: 当前版本主要实现图像处理算法，输出为处理后的关键帧图片。后续可扩展为视频处理。

### Q: 会损坏原始视频吗？
A: 不会，原始视频文件不会被修改，所有处理都在内存中进行。
