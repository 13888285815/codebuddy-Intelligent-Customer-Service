📖 使用说明

  1. Web 版：直接点击"打开客服界面"按钮，体验完整的客服功能。
  2. API 版：使用上方的 API 测试工具，或查看 versions/api/README.md 获取完整文档。
  3. Agent 版：访问 Agent 服务健康检查，或查看 versions/agent/README.md 了解更多功能。
  4. Widget 版：点击"查看演示"获取集成代码，参考 versions/widget/README.md 进行配置。
  5. Skill 版：点击"查看集成示例"选择目标平台，参考 versions/skill/README.md 进行集成。
  6. App 版：目前正在开发中，敬请期待！

详细文档请查看：versions/README.md 和 DEPLOYMENT.md
