## 智能客服工具

今天为用户创建了一个功能完善的智能客服 Agent Web 应用。

### 项目位置
`/Users/zzx/CodeBuddy/Claw/`

### 主要功能

1. **多轮对话**：支持上下文保持的连续对话
2. **FAQ 知识库**：内置常见问题解答
3. **意图识别**：自动识别用户意图（退款/订单/技术支持/其他）
4. **自动转人工**：连续3次未解决或用户主动请求转人工
5. **对话历史持久化**：SQLite 数据库存储
6. **管理后台**：
   - 登录验证（账号：admin，密码：admin123）
   - 查看对话记录
   - 满意度统计
   - AI 配置

### AI 大模型集成

1. **演示模式**（默认启用）：无需 API Key 即可体验智能对话
2. **多模型切换**：支持 GPT、Claude、文心一言、通义千问
3. **真实 AI 接入**：配置 API Key 后使用真实 AI 回答

### 安全功能

1. **对话内容加密**：AES-256-CBC 加密存储
2. **SQL 注入防护**：输入 sanitization
3. **XSS 防护**：特殊字符转义
4. **会话管理**：Token 验证
5. **密码加密**：SHA-256 哈希存储

### 用户端功能

1. **文件上传**：支持图片、视频发送
2. **常用工具**：
   - 表情选择器
   - 常用语快捷回复
   - 语音输入
   - 屏幕截图
   - 复制会话 ID
3. **响应式设计**：支持 Windows、iOS、Android、鸿蒙等所有设备

### 项目文件

- `server.js` - Express 后端服务
- `package.json` - 项目依赖
- `SPEC.md` - 规格说明文档
- `public/index.html` - 客服对话界面
- `public/admin.html` - 管理后台
- `public/style.css` - 共享样式

### 访问地址

- 客服界面：http://localhost:3000
- 管理后台：http://localhost:3000/admin
